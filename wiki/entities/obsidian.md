# Obsidian

## Overview

Obsidian is a markdown-based knowledge management application that serves as the browsing interface for the LLM Wiki. It provides graph view, wikilink navigation, and plugin support (Dataview, Marp) that complement the LLM-maintained wiki (source: [[llm-wiki-concept|LLM Wiki]]).

## Key Details

- Functions as the "IDE" for the wiki — the LLM writes, the user browses in Obsidian (source: [[llm-wiki-concept|LLM Wiki]])
- Supports `[[wikilinks]]` natively for cross-referencing between pages (source: [[llm-wiki-concept|LLM Wiki]])
- **Graph view** shows wiki topology — what's connected to what, which pages are hubs, which are orphans (source: [[llm-wiki-concept|LLM Wiki]])
- **Obsidian Web Clipper** browser extension converts web articles to markdown for the raw source collection (source: [[llm-wiki-concept|LLM Wiki]])
- **Attachment folder** can be set to `raw/assets/` so downloaded images land in the right place (source: [[llm-wiki-concept|LLM Wiki]])
- Plugins like **Dataview** run queries over YAML frontmatter, and **Marp** generates slide decks from markdown (source: [[llm-wiki-concept|LLM Wiki]])
