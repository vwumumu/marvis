# Pi

## Overview
Pi is an AI coding agent that comes bundled with OpenClaw. It runs in RPC mode by default, providing per-sender sessions without requiring additional configuration (source: [[openclaw-source|OpenClaw]]).

## Key Details
- **Role:** Default AI coding agent bundled with OpenClaw (source: [[openclaw-source|OpenClaw]])
- **Execution mode:** RPC mode with per-sender sessions (source: [[openclaw-source|OpenClaw]])
- **Capabilities:** Tool use, sessions, memory (source: [[openclaw-source|OpenClaw]])
